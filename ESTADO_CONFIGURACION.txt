================================================================================
                        SOLUCIÓN DE PROBLEMAS
================================================================================

PROBLEMA: "python no se reconoce como comando"
SOLUCIÓN: Python no está instalado o no está en PATH
          Ver: INSTALAR_PYTHON.md

PROBLEMA: No se puede acceder desde la red
SOLUCIÓN: 1. Configurar firewall: .\configurar_firewall.ps1
          2. Verificar que el servidor esté corriendo
          3. Verificar IP con: ipconfig

PROBLEMA: Error al conectar con BioStar
SOLUCIÓN: 1. Verificar IP en .env (BIOSTAR_HOST)
          2. Verificar credenciales en .env
          3. Probar conectividad: ping 10.0.0.100
